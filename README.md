# 百度小程序-第三方、工具类

提供百度小程序调用-各类工具


## 使用方式 1
现已将项目部署发布到Maven中央仓库 直接引入即可

      <dependency>
          <groupId>com.barcke.y</groupId>
          <artifactId>baidu-applet-spring-boot-start</artifactId>
          <version>0.0.1</version>
      </dependency>

## 使用方式 2 
先将两个模块install到本地项目中后 ----
引入下方依赖即可 ----

  

    <dependency>
          <groupId>com.barcke.y</groupId>
          <artifactId>baidu-applet-spring-boot-start</artifactId>
